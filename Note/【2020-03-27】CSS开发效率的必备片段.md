# CSS开发效率的必备片段

### 1、垂直水平居中

- **绝对定位方式且已知宽高**

```css

position: absolute;
top: 50%;
left: 50%;
margin-top: -3em;
margin-left: -7em;
width: 14em;
height: 6em;
```

- **绝对定位 ＋ 未知宽高 ＋ translate **

```css
position: absolute;
left: 50%;
top: 50%;
transform: translate(-50%, -50%);
//需要补充浏览器前缀
```

- **flex 轻松搞定水平垂直居中(未知宽高) **

```css
display: flex;
align-items: center;
justify-content: center;
```



### 2、文本末尾添加省略号

当文本的内容超出容器的宽度的时候，我们希望在其默认添加省略号以达到提示用户内容省略显示的效果。 

**宽度固定，适合单行显示...**

 ```css
overflow: hidden;
text-overflow: ellipsis;
white-space: nowrap;
 ```

**宽度不固定，适合多行以及移动端显示 **

```css
overflow: hidden;
text-overflow: ellipsis;
display: -webkit-box;
-webkit-line-clamp: 3;
-webkit-box-orient: vertical;
```



### 3、制造文本的模糊效果

```css
color: transparent;
text-shadow: 0 0 2px rgba(0, 0, 0, 0.5);
```
